# travel-app

A travel-app that is made in Vue with emphasis on Vue-Router implementation

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
